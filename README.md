## Dependencies

### Essential
```
grep
curl
sed
ffmpeg
git
```

### Optional
```
mpv - The default video player (recommended)
vlc - An alternative video player
```
  
## Install

### Linux / Mac
```sh
git clone https://github.com/kit-la/esp-anicli.git
cd esp-anicli
sudo make
```
